# Simulações - Doutorado em Ciências Climáticas

Repositório para armazenar simulações, modelos e pesquisas do doutorado em Ciências Climáticas.

## Estrutura

- `modelos/`: Modelos climáticos e de simulação
- `dados/`: Dados utilizados nas simulações
- `resultados/`: Saídas e resultados das simulações
- `scripts/`: Códigos para processamento e análise
